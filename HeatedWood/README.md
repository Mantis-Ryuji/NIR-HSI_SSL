## 方針
1. 再測定
2. MAEで学習
3. `(species, T, t)` から $z$ を当てる `estimator` を作成
4. $z$ → Decoder（MAEで学習）の `recon_loss` を通じて `estimator` を学習
5. 精度を通じてMAEのハイパラをチューニング（ベイズ最適化, `optuna`）$^{\text(1)}$
6. `estimator` の精度と補助指標を、Ref(SNV)空間とMAE潜在空間とで比較して有効性を主張
7. 劣化のシミュレーション精度を検討（条件の逆推定の精度を評価）
8. PGAを通じて熱処理による劣化の指標を作成（これを劣化の指標として使ってもよいかを検証） $^{\text(2)}$
9. 古材に適用（セグメンテーション＋解釈＋補助指標の評価）
10. ここでスコア使ってなんかする？（近赤外領域では樹種差・個体差・ノイズは劣化より小さくなっているからスコアとして使えるはず。← 熱処理結果で保証）

> (1): うまく再構成できているなら樹種・温度・時間の情報が上手く `z` に埋め込まれているはず。Ref(SNV)空間はこれらが絡みついて観測されるはずで、これが改善するとうれしい。
